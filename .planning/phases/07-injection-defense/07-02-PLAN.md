# Plan 07-02: internal/detector package

## Objective
Implement `internal/detector` — three-layer injection detection: pattern matching, encoding anomaly, and cosine-similarity outlier scoring.

## Requirements
SEC-04, SEC-05, SEC-06, SEC-07, SEC-08, SEC-09

## Tasks

- [x] Create `internal/detector/detector.go`
- [x] Define `Finding`, `Report`, `Category` types
- [x] Define constants: `DefaultDetectionThreshold = 0.70`, `DefaultOutlierThreshold = 0.85`
- [x] Implement `DetectPatterns(text string) []Finding` — 6 categories, 16 compiled regexes:
  - direct-override: 3 patterns (ignore/disregard/forget previous instructions)
  - role-injection: 3 patterns (you are now, act as, your new role)
  - delimiter-injection: 5 patterns (`<system>`, `---BEGIN PROMPT---`, `[INST]`, `|im_start|`, `###Instruction`)
  - role-prefix: 1 pattern (System:/Assistant:/Human: prefix lines)
  - jailbreak: 3 patterns (DAN mode, developer mode, pretend no restrictions)
  - exfiltration: 3 patterns (repeat/print/reveal above, what are your instructions, print your system prompt)
- [x] Implement `DetectEncoding(text string) []Finding` — 4 sub-detectors:
  - base64: regex match → length/4 validation → entropy gate (>4.5 bits/char) → decode verification
  - hex-escape: `\x41\x42...` sequences (4+ bytes)
  - hex-string: raw hex ≥32 chars with entropy >3.5 and length >40 (filters UUIDs)
  - ctrl-char-density: proportion of non-whitespace control chars; score = min(density×50, 0.90)
- [x] Implement `DetectOutliers(sentences []string, simMatrix [][]float64, threshold float64) []Finding`
  - outlier_score(i) = 1 - mean(simMatrix[i][j] for j ≠ i)
  - takes pre-normalization cosine similarity matrix from LexRank
- [x] Implement `Analyze(text string) Report` — combines DetectPatterns + DetectEncoding; outliers handled separately
- [x] Implement `shannonEntropy(s string) float64` — per-char entropy in bits
- [x] Write 28 unit tests covering all pattern categories, encoding types, and outlier computation
- [x] Wire `golang.org/x/text` for NFKC (already available from 07-01)
