**Unexported helper functions** (all in textrank.go):

1. `wordOverlapSim(s1, s2 []string) float64` — per Mihalcea & Tarau 2004:
   - If `len(s1) <= 1 || len(s2) <= 1`, return 0.0 (edge case: log(1)=0 causes division by zero)
   - Build set from s1, count common words with s2
   - If common == 0, return 0.0
   - Return `float64(common) / (math.Log(float64(len(s1))) + math.Log(float64(len(s2))))`

2. `trRowNormalize(matrix [][]float64)` — same logic as LexRank's rowNormalize but with a distinct name to avoid conflicts if both plans run in parallel:
   - Each row sums to 1.0; zero rows get uniform `1/n`

3. `powerIterateDamped(matrix [][]float64, damping, epsilon float64, maxIter int) []float64`:
   - Initialize scores to uniform `1/n`
   - `base := (1.0 - damping) / float64(n)`
   - Iterate: `next[i] = base + damping * sum_j(matrix[j][i] * scores[j])`
   - Converge when L1 diff < epsilon
   - Safety cap at maxIter

4. `trSelectTopN(scores []float64, n int, sentences []string) []string` — same logic as LexRank's selectTopN with distinct name:
   - `sort.SliceStable` descending by score
   - Take top n, `sort.Ints` to restore document order (SUM-05)

**Exported method:**

`func (t *TextRank) Summarize(text string, n int) ([]string, error)`:
1. `sentences := TokenizeSentences(text)` — return nil,nil if empty
2. `if n > len(sentences) { n = len(sentences) }` (SUM-04)
3. Tokenize each sentence into words via `tokenizeWords`
4. Build n*n word overlap similarity matrix using `wordOverlapSim`
5. `trRowNormalize(matrix)`
6. `scores := powerIterateDamped(matrix, textRankDamping, textRankEpsilon, textRankMaxIter)`
7. `return trSelectTopN(scores, n, sentences), nil`
