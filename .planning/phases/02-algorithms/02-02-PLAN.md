**Unexported helper functions** (all in lexrank.go):

1. `buildVocabAndIDF(sentences [][]string) ([]string, []float64)`:
   - Build `map[string]int` for document frequency (df) — count each word once per sentence using a `seen` set
   - Extract keys into `[]string`, `sort.Strings(vocab)` for determinism
   - IDF formula: `math.Log(float64(N) / float64(df[w]))` where N = len(sentences)
   - Return sorted vocab slice and parallel IDF `[]float64`

2. `buildTFVector(words []string, wordIdx map[string]int, vocabSize int) []float64`:
   - Count word occurrences in `words`
   - TF = count / total words in sentence
   - Return `[]float64` of length `vocabSize`, indexed by `wordIdx`

3. `idfCosine(v1, v2, idf []float64) float64`:
   - IDF-modified cosine: `sum(idf[i]^2 * v1[i] * v2[i]) / (sqrt(sum(idf[i]^2 * v1[i]^2)) * sqrt(sum(idf[i]^2 * v2[i]^2)))`
   - Return 0.0 if either norm is 0 (avoid NaN)

4. `rowNormalize(matrix [][]float64)`:
   - Each row sums to 1.0; if row sum is 0, assign uniform `1/n`

5. `powerIterate(matrix [][]float64, epsilon float64, maxIter int) []float64`:
   - Initialize scores to uniform `1/n`
   - Iterate: `next[i] = sum_j(matrix[j][i] * scores[j])`
   - Converge when L1 diff < epsilon
   - Use epsilon=0.0001, maxIter=1000

6. `selectTopN(scores []float64, n int, sentences []string) []string`:
   - Create `[]scored{idx, score}` pairs
   - `sort.SliceStable` descending by score (deterministic tie-breaking)
   - Take top n indices, `sort.Ints(top)` to restore document order (SUM-05)
   - Return `sentences[idx]` in document order

**Exported method:**

`func (l *LexRank) Summarize(text string, n int) ([]string, error)`:
1. `sentences := TokenizeSentences(text)` — return nil,nil if empty
2. `if n > len(sentences) { n = len(sentences) }` (SUM-04)
3. Tokenize each sentence into words via `tokenizeWords`
4. `buildVocabAndIDF` on word lists
5. Build `wordIdx map[string]int` from sorted vocab
6. Build TF-IDF vectors for each sentence
7. Build n*n cosine similarity matrix using `idfCosine` (continuous — no threshold per research recommendation)
8. `rowNormalize(matrix)`
9. `scores := powerIterate(matrix, 0.0001, 1000)`
10. `return selectTopN(scores, n, sentences), nil`

Constants: `const lexrankEpsilon = 0.0001` and `const lexrankMaxIter = 1000`
