**7. Algorithms**

Brief explanation of each:

_LexRank (default):_ Graph-based algorithm using TF-IDF cosine similarity. Sentences are ranked by eigenvector centrality — sentences that are most similar to the most other sentences rank highest.

_TextRank:_ Graph-based algorithm using word-overlap similarity (Jaccard-style). Ranks sentences by PageRank-style iteration. Often favors shorter, more distinct sentences.

_graph:_ Delegates to `github.com/didasy/tldr` as a baseline comparison.

**8. Algorithm comparison table**

| Property | LexRank | TextRank |
|----------|---------|----------|
| Similarity metric | TF-IDF cosine | Word overlap |
| Ranking method | Eigenvector centrality | PageRank damping |
| Best for | Long articles, dense technical content | Conversational text, transcripts |
| Deterministic | Yes | Yes |
